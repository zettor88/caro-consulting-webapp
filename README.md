# Caro Consulting Web App

Esta aplicación web fue generada con Next.js, Tailwind CSS y Recharts, siguiendo la especificación "Executive Dark Mode".

## Requisitos Previos

Para ejecutar este proyecto, necesitas tener instalado **Node.js** (versión 18 o superior).

1.  Ve a [nodejs.org](https://nodejs.org/)
2.  Descarga la versión "LTS" (Recomendada).
3.  Instálalo en tu computador siguiendo los pasos del instalador.

## Cómo Iniciar

Una vez instalado Node.js, abre tu terminal en esta carpeta (`Caro_Consulting/webapp`) y ejecuta:

1.  **Instalar dependencias**:
    ```bash
    npm install
    ```

2.  **Iniciar el servidor de desarrollo**:
    ```bash
    npm run dev
    ```

3.  Abre tu navegador en `http://localhost:3000`.

## Estructura

-   `/`: Landing Page pública (Servicios, Casos de Éxito).
-   `/login`: Pantalla de acceso simulada (cualquier credencial funciona).
-   `/dashboard`: Portal de Cliente con gráficos de rentabilidad y documentos.
